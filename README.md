# Blx-backend
API em Fast API para o cadastro de produtos, pedidos e usuarios
## Configurações Base
- Unix Ubuntu 20.04
- Python 3.8.10
- fastAPI 0.85.0
## Principais Ferramentas utilizadas:
- alembic 1.8.1
- SQLAlchemy 1.4.41
- pydantic 1.10.2
- pytest 7.1.3
- uvicorn 0.18.3
## Instalação
1. criar ambiente virtual python
2. clonar o plojeto
3. instalar o requirements.txt

    `pip install requirements.txt`
